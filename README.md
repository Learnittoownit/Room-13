#  Room 13

> A psychological horror game where every corridor hides a mystery.

Room 13 is a 3D horror game developed using **Unity Engine**. Set inside an abandoned haunted hospital, players must navigate dark hallways armed with only a flashlight while uncovering the terrifying secrets hidden within the infamous Room 13.

The game focuses on creating an immersive atmosphere through environmental storytelling, lighting design, and suspense-driven gameplay.

---

## ✨ Features

- 👻 Immersive first-person horror experience
- 🔦 Flashlight-based exploration
- 🏥 Haunted hospital environment
- 🧩 Environmental storytelling
- 🎮 Interactive gameplay mechanics
- 🌑 Dynamic lighting and atmosphere
- 🎯 Objective-driven level progression

---

## 🛠 Tech Stack

- Unity Engine
- C#
- 3D Level Design
- Lighting System
- Game Mechanics
- Unity Physics
- Visual Studio

---

## 🏗 Game Design

The game was designed around environmental immersion, using lighting, sound, and level layout to build tension without relying on excessive jump scares.

The hospital environment guides players naturally through exploration while revealing the mystery behind Room 13.
